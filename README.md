
# SwiftChess

**SwiftChess** is a lightweight, high-performance React chessboard built from scratch.  
It uses **SVG pieces**, minimal dependencies, and optimized rendering for smooth drag & drop gameplay.

---

## Features

- Fully interactive 8x8 chessboard  
- Drag & drop pieces with smooth SVG animations  
- Supports move highlights and premoves  
- Lightweight and fully customizable  
- No external heavy chessboard libraries  

---

## Installation

```bash
git clone https://github.com/RajanDhamala/SwiftChess.git
cd SwiftChess
npm install
npm start
